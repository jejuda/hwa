# 🗓️ 디스코드 보스 타임봇 (보탐봇)

디스코드에서 게임 내 보스 몬스터의 처치(컷) 시간을 기록하고, 다음 젠 시간을 계산하여 자동으로 채널에 알림을 전송해 주는 봇입니다. 5분 전에는 음성 채널에 TTS(Text-to-Speech)로 음성 안내를 발송해 줍니다.

## 🛠️ 주요 기능
- **보스 관리**: `/보스등록`, `/보스삭제`, `/보스수정`
- **시간 기록**: `/컷` (처치 시간 기록), `/젠` (다음 예정 시간 강제 입력), `/컷취소` (실수 기록 롤백)
- **현황 조회**: `/보스목록` 또는 `/보탐` (남은 시간순 자동 정렬 리스트)
- **텍스트 채널 지정**: `/알림채널설정`으로 봇 알림을 수신할 텍스트 채널을 지정
- **음성 채널 TTS 알림 (5분 전)**: `/음성채널설정`으로 지정한 음성 채널에 봇이 상시 접속하여 보스 젠 5분 전에 한국어 TTS로 리포트
- **스마트 시간 인식**: `14:30`, `1430` 등 형식의 절대 시간 및 `10`, `10분전` 형태의 상대 시간 및 하루단위 롤오버 보정 지원
- **부분 이름 매칭**: `/컷 가르` 형태로 입력해도 `가르투아`를 자동으로 인식해 동작하는 고품질 UX 지원
- **기본 보스 연동**: `가르투아`, `구루타`, `쉬라크`, `라그타`, `타르탄`, `카샤파`, `노블루드`, `비슈베다`, `카루카`, `바르시엔`, `악시오스` 등 11종 기본 프리셋 자동 탑재

## 🚀 구동 및 설정 방법

### 1. 디스코드 개발자 포털 설정
1. [Discord Developer Portal](https://discord.com/developers/applications)에 접속하여 본인의 애플리케이션을 선택합니다.
2. **Bot** 탭으로 이동한 뒤, **Reset Token**을 눌러 봇 토큰을 복사합니다.
3. **OAuth2** -> **URL Generator** 탭으로 이동합니다.
4. **Scopes**에서 `bot` 및 `applications.commands`를 체크합니다.
5. **Bot Permissions**에서 다음 권한을 체크합니다.
   - `Read Messages/View Channels`
   - `Send Messages`
   - `Embed Links`
   - `Connect` (음성 채널 연결 권한)
   - `Speak` (음성 송출 권한)
6. 하단에 생성된 URL로 접속하여 봇을 디스코드 서버에 초대합니다.

### 2. 프로젝트 설정
1. 프로젝트 루트의 `.env` 파일을 메모장 등으로 열고 복사해둔 봇 토큰을 적어 넣습니다:
   ```env
   DISCORD_TOKEN=실제_봇_토큰_값
   CLIENT_ID=1520011332630351882
   ```

### 3. 패키지 설치 및 실행
1. 터미널 또는 PowerShell을 열고 프로젝트 폴더로 이동합니다.
2. 디스코드 API에 슬래시 명령어를 등록합니다:
   ```bash
   npm run deploy
   ```
3. 봇을 구동합니다:
   ```bash
   npm start
   ```

## 🎮 명령어 설명 및 사용법 예시
- `/알림채널설정 [채널(선택)]`
  - 알림이 발송될 텍스트 채널을 지정합니다. 생략하면 현재 채널로 자동 설정됩니다.
- `/음성채널설정 [채널(선택)]`
  - 5분 전 TTS 알림을 송출할 음성 채널을 지정합니다. 생략하면 현재 본인이 들어가 있는 음성 채널로 봇이 입장합니다.
- `/음성채널해제`
  - 설정을 초기화하고 음성 채널에서 봇을 퇴장시킵니다.
- `/보스등록 이름:바포메트 젠주기:4 메모:정상에 리젠`
  - 4시간 주기의 바포메트 보스를 등록합니다.
- `/컷 이름:가르`
  - 부분 일치 검색으로 `가르투아`를 자동으로 찾아 현재 시간 기준으로 처치 기록을 남깁니다.
- `/컷 이름:가르 시간:10분전`
  - 10분 전에 처치한 것으로 계산하여 기록을 남깁니다.
- `/젠 이름:라그타 시간:23:40`
  - `라그타` 보스의 다음 젠 시간을 오늘 23시 40분으로 강제 고정합니다. (만약 현재 시각보다 15분 이상 과거인 경우 내일 23:40분으로 자동 처리됩니다)
- `/컷취소 이름:가르`
  - 직전에 입력한 기록을 실수로 오기입했을 경우, 기록을 이전 값으로 롤백합니다.
- `/보스목록` 또는 `/보탐`
  - 등록된 보스 리스트를 젠 임박 순으로 예쁜 카드(Embed) 형태로 출력합니다.
